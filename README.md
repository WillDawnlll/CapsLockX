# CapslockX

## 使用教程第一

### 鼠标控制
wasd = 鼠标平滑移动
rf   = 垂直平滑滚轮（在chrome下有时失灵，原因不明)
zc   = 水平平滑滚轮

`    = 回车键

u    = 撤销
U    = 重做
hljk = 上下左右键（跟 vim 一样）

b = Delete

qe   = 鼠标右、左键


g    = 复制当前选择，并用google搜索

F5   = 暂停
F6   = 上一首
F7   = 下一首
F8   = 停止

F10  = 静音
F11  = 音量减
F12  = 音量加
