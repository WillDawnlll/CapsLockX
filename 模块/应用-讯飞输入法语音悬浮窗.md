效果如下

| 模式 | 按键| 功能说明 |
| - | :-: | - |
| 任意 | Win + H | 启动/切换讯飞语音输入 |
*注1：原 Win+H 的功能是 Windows 自带听写，安装本插件后会变成 Win+Shift+H
*注2：若没有安装讯飞语音则会询问是否下载安装
